# sistema-rutas-urbanas-grafos
# Sistema Inteligente de Rutas Urbanas para E-bikes

Proyecto desarrollado en Python utilizando teoría de grafos para calcular rutas óptimas mediante el algoritmo de Dijkstra y árboles de expansión mínima con Kruskal.

## Tecnologías
- Python
- NetworkX
- Matplotlib

## Funcionalidades
- Visualización de grafos
- Cálculo de caminos mínimos
- Simulación de rutas urbanas
- Restricciones de circulación
- Árbol de expansión mínima (Kruskal)

## Algoritmos utilizados
- Dijkstra
- Kruskal
- Haversine
